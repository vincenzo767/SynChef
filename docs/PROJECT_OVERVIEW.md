# SynChef

[![Development Status](https://img.shields.io/badge/Status-In%20Development-yellow)]()
[![Full Stack](https://img.shields.io/badge/Stack-Full--Stack-blue)]()

AI-Powered Real-Time Global Cooking Assistant

## Overview

SynChef revolutionizes home cooking with parallel timers, 3D global recipe exploration, AI-powered ingredient substitutions, and a beautiful Focus Mode UI.

## Quick Start

See [README.md](README.md) for detailed setup instructions.

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Database
```sql
CREATE DATABASE synchef_db;
```
